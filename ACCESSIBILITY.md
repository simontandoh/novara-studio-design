# Accessibility Notes

## Key Decisions
- Added consistent `:focus-visible` outlines for interactive elements.
- Implemented reduced motion support to honor user preferences.
- Improved contact form labels, error associations, and required field handling.
- Ensured heading hierarchy remains consistent across pages.
- Added ARIA labeling for the navigation overlay and icon buttons.

## Known Limitations
- Contrast is tuned for the current brand palette. Re-test if colors shift.
- The portfolio carousel uses horizontal scrolling; keyboard navigation is available via native scroll.

If you spot any issues, email hello@novarastudios.co.uk.
